PWA Demo

In a full deployment, register a service worker for the app shell and pass its registration into registerBackgroundSync so queued progress syncs even after the tab is closed: 

navigator.serviceWorker.register('/sw.js').then((reg) => {
  const bridge = new CasuyaBridge({ apiBaseUrl: 'https://api.casuya.co.tz' });
  bridge.runtime.syncEngine; // wired automatically by bridge.start()
  registerBackgroundSync(bridge.runtime.bus, bridge.runtime.syncEngine, {
    swRegistration: reg,
  });
});
